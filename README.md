# In-Memory-of-Medical-Students-Who-passed-Away
A  memorial HTML page for young doctors, residents and medical students who passed away due to overwork, training fatigue or unexpected incidents. 致因过劳、规培压力与意外离世的年轻医生、规培医师及医学生：谨以此网页纪念，愿他们被铭记，愿行业更善待每一位医者。
